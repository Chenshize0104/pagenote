# Keyword

This is an example component.

```jsx
import { Keyword } from 'pagenote';

export default () => <div>
  <Keyword>key</Keyword>
  <key-word>key</key-word>
  </div>
```

```javascript
const a = document.createElement('div');
a.innerHTML = '<key-word>key</key-word>'
document.body.append(a)
```
